## Prerequisites

1. Install [Bun](https://bun.sh) globally

   ```console
   foo@bar:~$ curl -fsSL https://bun.sh/install | bash
   ```

2. Install dependencies

   ```console
   foo@bar:video-player$ bun i
   ```

## Scripts

### Start development server

```text
bun run dev
```

### Build for production

```text
bun run build
```

### Serve for production

```text
bun start
```
